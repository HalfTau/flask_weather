# Flask Weather

A small weather web app built with **Python, Flask, and the OpenWeather API**.

## Features

- Search for a city and see:
  - Current conditions (temperature, feels like, humidity, description)
  - 7-day forecast with daily highs and lows
  - Next 24 hour timeline
- Bootstrap-based layout
- Localized timestamp for America/Los_Angeles
- API keys handled via environment variables

## Tech stack

- Python 3.x
- Flask
- OpenWeather One Call API 3.0
- HTML + Jinja templates
- Bootstrap 5
